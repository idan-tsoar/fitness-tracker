# Fitness Tracker 🏋️

אפליקציית מעקב דיאטה אישית — עברית, RTL, ללא תלויות.

## תכונות

- **מעקב יומי** — משקל, קלוריות, חלבון, אימונים והערות לכל יום
- **ממוצעים שבועיים אוטומטיים** — כולל חישוב שינוי במשקל מול שבוע קודם
- **תפריט שבועי מותאם אישית** — 7 ימים, מגוון בשר/עוף/דגים, סגנונות ישראלי/אסיאתי/גריל
- **שייק בוקר קבוע** — שיבולת שועל, אבקת חלבון, חמאת בוטנים, אוכמניות
- **גרפים** — משקל, קלוריות, חלבון ואימונים לאורך זמן (Chart.js)
- **הוספת שבוע בלחיצה** — כפתור ➕ מוסיף שבוע חדש עם תאריכים מחושבים
- **שמירה אוטומטית** — כל הנתונים ב-localStorage של הדפדפן
- **ייצוא/ייבוא JSON** — גיבוי מלא של הנתונים

## מבנה

```
fitness-tracker/
├── index.html     מבנה האפליקציה
├── style.css      עיצוב RTL + responsive
├── app.js         לוגיקה, חישובים וגרפים
├── data.json      נתונים התחלתיים ותפריט שבועי
└── README.md
```

## איך להריץ מקומית

פתח את `index.html` בדפדפן — או העדף להריץ דרך שרת סטטי כדי ש-`data.json` ייטען:

```bash
python3 -m http.server 8000
# ואז נכנסים ל- http://localhost:8000
```

## פריסה ל-GitHub Pages

3. Branch: `main` / `root`
